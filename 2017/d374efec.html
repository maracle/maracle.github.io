notepad++ 一直报错here are still some pending actions

There are still some pending actions to complete installing or removing some plugins.

其他答案对我不起作用，因为这些文件不存在。 但可以通过everything找到他们。

先彻底关闭notepad++，然后用everything搜索病删除以下文件，再重启，即可恢复notepad++的正常功能。

删除文件和文件夹：

  • PluginManager.ini
  • PluginManagerPlugins.xml
  • PluginManagerPlugins.zip
  • plugin_install_temp
