Applications

这里分享一些我常用的APP，手机端和电脑端的都有，日常更新。

Android

手机端的软件其实也没有用很多，但以下这些还是常用的，也是比较优质的，值得推荐！具体的链接就不贴了，大家可以在Google Play上找到。

  • 纯纯写作，手机端比较好的编写利器，但是订阅制了，有点贵，支持markdown，latex，富文本等。

  • Fv悬浮球 极简单手操作，一个手势即可完成复杂操作强大小小的体积，容纳了浏览器，文件管理器，悬浮窗，搜索引擎，应用管理，手写识别等强大功能，等你来

  • 欧陆词典，全平台适配，扩展词库超多，高级功能需要付费，尤其PC端不付费不能使用！

  • OneDrive，网盘，全平台适配，配合office的1TB空间就不错。

  • Resplash 壁纸软件，unsplash网站的第三方客户端。

  • V2er v2ex的第三方客户端。

  • openhub for github Github的第三方客户端，很好用。

  • Telegram 不解释。

  • Google Voice 申请了2个GV号码，用来注册乱七八糟的东西，接收验证码。

  • via浏览器，可自定义，自由度高，体积小。

  • edge浏览器 配合PC同步书签，微软新上架的，经常需要联动PC和手机的非常适合。

  • Quark夸克浏览器 UC出品的浏览器，目前主要用的。

  • Reddit 程序员网站？

  • StackOverflow 程序员问答网站。

  • Quora 类似国内的知乎。

  • Medium 优质博客平台，需要付费。

  • MX Player 视频播放器，常用。

  • nPlyaer iOS平台的视频播放神器，已经移植到了安卓上，非常的方便，支持WiFi播放，自带杜比音效。

  • VLC 跨平台制霸视频播放器，安卓端的使用体验也不错，不过不常用。

  • Coolapk 酷安市场，搞机人士聚集地，各种开发者。

  • 湾区日报 很不错的app，我很喜欢。

  • 虎扑 各种游戏和影视剧交流，还有知名的步行街。男人和男人的交流平台（当然也有女士）。

  • NGA 是国内专业的游戏玩家社区。

  • 小木虫 学术科研互动平台，会员主要来自国内各大院校、科研院所的博硕士研究生，企业研发人员，这里拥有旺盛的人气、良好的交流氛围及广阔的交流空间。

  • 微信读书 跨平台读书神器，参与打卡、签到、组队等等活动，能够免费白嫖，关键是支持各种平台，包括墨水屏阅读器，例如kindle。

  • 静读天下专业版 书源多，排版舒适，非常适合喜欢看文的。

  • 咪咕音乐 中国移动加持的音乐app，能够实现白嫖周杰伦的歌曲，非常舒适。

  • 滴答清单 时间和任务规划神器，在安卓和ios都支持非常好，界面美观，提醒功能放贴心。付费订阅模式。

  • 墨墨背单词 高效抗遗忘，轻松规划海量词汇记忆。能够根据记忆曲线不断记忆，界面简洁明了，背单词神器。付费买单词数量。

  • 词根词缀词典 词根字典致力于打造最实用的英语单词以及词根词缀查询系统！

  • 锤子便签 可能是史上最漂亮的便签应用，你或许会因它重新喜欢上记录和表达。可以分享为图片，支持富文本和markdown。能多端同步。

  • 向日葵远程控制 手机控制电脑的不错选择。

  • 古诗文网 除了西窗烛，可能就是这款了，用来看诗句背诗。免费。

  • 西窗烛 界面优美，古诗文爱好者聚集地，非常多的朗读和写字，不过越来越臃肿了，高级功能需要付费。

  • BIilibili 二次元和鬼畜的聚集地，应该没有不知道的人了。

  • 倒带 本地音乐播放器，还是不错的。更重要的，支持网易账号登陆，播放网易收藏的音乐和云盘音乐，神器。

  • musicolet 本地音乐播放器，非常喜欢的界面简洁好看。

  • 轻听 本地音乐播放器，安卓上效果不错的。

  • 厚墨 极简设计，纯粹阅读。无广告，无推送。支持.epub/.mobi/.azw3格式文件。支持添加JsonPath CSS Query指定规则的书源。支持付费订阅使用云端相关功能。

  • 央视频 短视频国家队，强大的直播工具，支持央视和地方卫视直播、录播，聚合高清4K，无广告，无付费。

  • 龙卷风收音机 爱好听收音机的最佳app吧？不仅有央广还有个地方的广播。

  • 讯飞语记 讯飞语记由科大讯飞出品，是一款语音变文字输入的云笔记APP，并支持录音速记、图文编排、任务提醒、朗读笔记、多端同步、分类管理等功能。

  • shareme ShareMe - An ad-free P2P file transfer tool that works without connecting to the internet.More than 200 million users are using it. Just ask for it. ShareMe!!

macOS

苹果端的软件还真不少，从娱乐到开发到写作都有，总之应用生态很好，这也是我喜欢mac的原因之一了。

当然了，mac用户也可以参考https://github.com/jaywcjlove/awesome-mac和https://github.com/iCHAIT/awesome-macOS。

下面一些是我常用但不一定能找得到软件，分享出来。那些类似QQ和微信这样可以找得到的就不列出来了。

  • Develop
    • JetBrains JetBrains全家桶开发工具，不用解释。
    • XCode 苹果家自带的，不解释。
    • VS Code 微软家唯一在用的产品，也许是的。
    • Sublime Text 不解释。
    • Atom 不解释。
    • Vim 不解释，我现在用的很舒畅。
    • Docker 很方便的容器。
    • DBeaver 数据库软件，很棒。
    • Jupyter Notebook 很酷，自己查吧。
  • Terminal
    • Hyper JS写的命令行，用起来还是很溜的。
    • iTerm2 在Hyper之前一直使用的终端，现在依然在用。
  • Writting & Reading
    • Ulysses 优质的写作工具。
    • Notion 上面有介绍，非常好用。
    • MacDown mac下的markdown编辑器。
    • MacTeX mac下的LaTeX。
    • Typora 跨平台的markdown写作软件。
    • Joplin 跨平台的markdown写作软件。
    • Reeder RSS阅读器。
  • Tools
    • Handshaker mac和android手机传输数据的工具。
    • FinderGo Find拓展程序。
    • Dozer 隐藏多余的状态栏图标。
    • FiScript Find拓展程序。
    • Itsycal 日历。
    • Flux 屏幕护眼。
    • Keka 压缩软件。
    • eZip 第三方开发者独立作品，目前来说是我用过最好的压缩软件。
    • IINA 视频软件。
    • VLC 同上。
    • MPV 同上。
    • Skim 非常好用的一个pdf浏览器。
    • CotEditor 很好用的一款文本编辑器。
    • Motrix 全平台的下载软件，很好用。
    • balenaEtcher 启动盘制作工具，很强大。
    • Proxyee Down 跨平台下载器，强大。
    • Xnip 截屏软件，很好用。
    • Unsplash mac下的壁纸更换软件。
    • V2rayX V2ray的第三方客户端。
    • ShadowsocksX-NG 不解释。
  • Media
    • cocomusic Electron + Vue写的一个第三方音乐播放器。
    • 人人影视 非常好用的一个全平台看剧神器，但linux下没有。
  • Virtual Machine
    • VMware
    • Virtual Box
    • Parallels Desktop
  • Mail
    • Spark 界面美观，好用。
  • Browser
    • Vivaldi 跨平台的高度可定制化的浏览器。
    • ungoogled-Chromium 去除google特性的Chromium。
    • Librefox 隐私安全的Firefox。
  • Social
    • Telegram 不解释。
    • Whalebird 第三方的Mastodon客户端。
  • Cli Applications
    • carbon-now-cli 将代码生成图片。
    • docz 超级棒！
    • bat
    • m-cli
    • Mac-CLI
    • fine.sh
    • Google Drive
    • gitbook-cli
    • gitmoji-cli
    • fast-cli
    • leetcode-cli
    • splash-cli
    • rebound Check python error on internet.
